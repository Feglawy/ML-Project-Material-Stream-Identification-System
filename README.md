# ML Project: Material Stream Identification System


## project setup
1. create virtual enviroment

```shell
python -m venv .venv
```
2. activate the vertual enviroment
```shell
.venv\scripts\activate
```
3. install required libraries
```shell
pip install -r .\requirements.txt
```
